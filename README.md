EduTech Platform: Ciberseguridad Aplicada

EduTech Platform es una solución de aprendizaje diseñada para cerrar la brecha de talento en ciberseguridad en Colombia. Mediante una metodología práctica, gamificada y centrada en el usuario, permitimos que estudiantes y profesionales de TI obtengan certificaciones técnicas verificables con experiencia demostrable.  

🚀 Acerca del Proyecto

Este MVP ha sido desarrollado como respuesta a la necesidad de entornos donde los estudiantes no solo consuman teoría, sino que validen sus conocimientos mediante retos prácticos. El proyecto sigue un ciclo de vida iterativo-incremental, permitiendo una mejora continua basada en el feedback del usuario.  

🛠️ Metodología y Diseño

Design Thinking: Aplicamos un proceso de empatía para entender las frustraciones de los estudiantes frente a las certificaciones costosas y excesivamente teóricas.  
Gestión Ágil: Utilizamos Scrum para la gestión de sprints, lo que nos permite iterar funcionalidades clave como el sistema de evaluación y la certificación automática.  
Enfoque MVP: Nos enfocamos en las historias de usuario críticas: Registro Seguro y Obtención de Certificado.  

📋 Características Principales (RFs)

RF-01 Gestión de Usuarios: Sistema de autenticación con validaciones de seguridad.  
RF-03 Visualización de Contenido: Acceso a material multimedia y guías técnicas.  
RF-04 Evaluación Automática: Motor de calificación instantánea con umbral de aprobación del 80%.  
RF-05 Emisión de Certificados: Generación dinámica de certificados PDF con ID único.  

🏗️ Stack Tecnológico

Frontend: HTML5, CSS3, JavaScript (Vanilla).
Librerías: jsPDF (generación de certificados), Font Awesome (iconografía).
Arquitectura: Diseño responsivo y modular centrado en la usabilidad.  

📖 Instrucciones de Uso

Acceso: Inicia en index.html. El sistema valida contraseñas de +8 caracteres, mayúsculas y números.  
Exploración: Selecciona tu ruta técnica en catalog.html.  
Aprendizaje: Navega por los módulos: Teoría, Laboratorio y Examen.  
Certificación: Si superas el 80% de la nota, el sistema habilitará automáticamente la emisión de tu certificado profesional.
